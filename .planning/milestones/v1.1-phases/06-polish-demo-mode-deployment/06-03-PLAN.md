   - New approach — highlight-once, reveal-line-by-line:

     d. When `highlightedHtml` changes (new content highlighted), start a reveal interval:
        ```
        let count = 0;
        const totalLines = lines.length;
        const interval = setInterval(() => {

          setVisibleLineCount(count);
          if (count >= totalLines) clearInterval(interval);

        ```
