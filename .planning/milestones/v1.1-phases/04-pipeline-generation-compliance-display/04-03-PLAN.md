    **Data reading:**
    - Read `analysisResult.risk.topFindings` from Zustand — array of `{ finding: string, severity: string, affectedNodes: string[], framework?: string, recommendation: string }`. NOTE: `affectedNodes` is PLURAL and is an ARRAY of strings. `framework` is OPTIONAL (may be undefined).
    - Read `status` from Zustand.

    **Severity badges:**
    ```typescript
    const SEVERITY_COLORS: Record<string, string> = {
      critical: 'bg-red-500/20 text-red-400 border-red-500/30',
      high: 'bg-orange-500/20 text-orange-400 border-orange-500/30',
      medium: 'bg-amber-500/20 text-amber-400 border-amber-500/30',
      low: 'bg-blue-500/20 text-blue-400 border-blue-500/30',
      info: 'bg-emerald-500/20 text-emerald-400 border-emerald-500/30',
    };
    const SEVERITY_ORDER: Record<string, number> = { critical: 0, high: 1, medium: 2, low: 3, info: 4 };
    ```

    **Expandable rows:**
    - Track expanded row indices with `const [expandedRows, setExpandedRows] = useState<Set<number>>(new Set())`.
    - Summary row (always visible): severity badge + finding text (truncated to 1 line) + affected nodes (joined with comma: `finding.affectedNodes.join(', ')`).
    - Click row to toggle expanded state.
    - Expanded detail (below summary): framework (guard with `finding.framework ?? 'General'` since it's optional), recommendation, and any control references.
    - Use `ChevronRight` icon that rotates to `ChevronDown` when expanded (CSS `rotate-90` transition).

    **Filter dropdowns (top of table):**
    - Framework filter: Select dropdown with "All Frameworks" default + unique framework values. NOTE: `framework` is optional on findings, so collect unique frameworks with: `[...new Set(findings.map(f => f.framework).filter(Boolean))]`.
    - Severity filter: Select dropdown with "All Severities" default + severity levels.
    - Both use shadcn Select component.

    **Sort and filter with useMemo:**
    ```typescript
    const sortedFiltered = useMemo(() => {
      let rows = [...findings];
      if (frameworkFilter !== 'all') rows = rows.filter(r => (r.framework ?? '') === frameworkFilter);
      if (severityFilter !== 'all') rows = rows.filter(r => r.severity === severityFilter);
      rows.sort((a, b) => {
        const orderA = SEVERITY_ORDER[a.severity] ?? 5;
        const orderB = SEVERITY_ORDER[b.severity] ?? 5;
        return orderA - orderB; // severity descending (critical first)
      });
      return rows;
    }, [findings, frameworkFilter, severityFilter]);
    ```

    **Layout:**
    ```

      <div className="p-6">
        <div className="flex items-center justify-between mb-4">
          <h3 className="text-sm font-medium text-slate-400">Findings</h3>
          <div className="flex gap-2">
            {/* Framework filter */}
            {/* Severity filter */}
          </div>
        </div>
        <div className="space-y-1">
          {sortedFiltered.map((finding, idx) => (
            <div key={idx}>
              {/* Summary row — clickable */}
              <div onClick={() => toggleExpand(idx)}
                   className="flex items-center gap-3 px-3 py-2 rounded-md hover:bg-slate-700/50 cursor-pointer transition-colors">
                <ChevronRight className={cn('h-4 w-4 text-slate-500 transition-transform', expandedRows.has(idx) && 'rotate-90')} />
                <SeverityBadge severity={finding.severity} />
                <span className="text-sm text-slate-200 flex-1 truncate">{finding.finding}</span>
                <span className="text-xs text-slate-500">{finding.affectedNodes.join(', ')}</span>
              </div>
              {/* Expanded detail */}
              {expandedRows.has(idx) && (
                <div className="ml-10 px-3 py-2 text-xs text-slate-400 space-y-1 border-l-2 border-slate-700">
                  <p><span className="text-slate-500">Framework:</span> {finding.framework ?? 'General'}</p>
                  <p><span className="text-slate-500">Recommendation:</span> {finding.recommendation}</p>
                </div>
              )}
            </div>
          ))}
        </div>
      </div>
    </Card>
    ```

    **PipelinePreview component (`src/components/dashboard/pipeline-preview.tsx`):**

    Create a `'use client'` component.

    **Data reading:**
    - Read `analysisResult.pipeline` from Zustand — `PipelineConfig` type with `githubActions`, `securityScanning`, `infrastructureAsCode`.
    - Read `status` from Zustand.

    **Shiki syntax highlighting:**
    - Import `codeToHtml` from `'shiki/bundle/web'` (NOT from `'shiki'` — avoids 1MB bundle).
    - Use `useEffect` to pre-compute highlighted HTML for all 3 tabs when `pipelineConfig` arrives:
      ```typescript
      const [highlightedHtml, setHighlightedHtml] = useState<Record<string, string>>({});

      useEffect(() => {
        if (!pipelineConfig) return;

        async function highlight() {
          const [ghHtml, secHtml, infraHtml] = await Promise.all([
            codeToHtml(pipelineConfig.githubActions.yaml, {
              lang: 'yaml',
              theme: 'github-dark',
            }),
            codeToHtml(
              pipelineConfig.securityScanning.tools.map((t: { name: string; config: string }) => `# ${t.name}\n${t.config}`).join('\n---\n'),
              { lang: 'yaml', theme: 'github-dark' }
            ),
            codeToHtml(pipelineConfig.infrastructureAsCode.config, {
              lang: pipelineConfig.infrastructureAsCode.provider === 'terraform' ? 'hcl' : 'yaml',
              theme: 'github-dark',
            }),
          ]);
          setHighlightedHtml({ github: ghHtml, security: secHtml, infra: infraHtml });
        }

        void highlight();
      }, [pipelineConfig]);
      ```
    - Do NOT store highlighted HTML in Zustand — it's ephemeral display data.

    **Tabs:**
    - Import `Tabs`, `TabsContent`, `TabsList`, `TabsTrigger` from `@/components/ui/tabs`.
    - Track active tab with `const [activeTab, setActiveTab] = useState('github')`.
    - Three tabs: "GitHub Actions", "Security Scanning", "Infrastructure".
    - Tab triggers styled with dark theme: `data-[state=active]:bg-slate-700 text-slate-400 data-[state=active]:text-slate-100`.
    - TabsList styled: `bg-slate-900 border border-slate-700`.

    **Code display:**
    - Render highlighted HTML via `dangerouslySetInnerHTML={{ __html: highlightedHtml[activeTab] ?? '' }}`.
    - Container: `text-xs font-mono overflow-auto max-h-96 [&_pre]:p-4 [&_pre]:rounded-lg [&_pre]:bg-transparent [&_code]:leading-relaxed`.
    - Show `<Skeleton />` while shiki is loading (highlightedHtml is empty).

    **Copy to clipboard:**
    ```typescript
    const [copied, setCopied] = useState(false);

    const getRawContent = (tab: string): string => {
      if (!pipelineConfig) return '';
      switch (tab) {
        case 'github': return pipelineConfig.githubActions.yaml;
        case 'security': return pipelineConfig.securityScanning.tools.map(t => t.config).join('\n---\n');
        case 'infra': return pipelineConfig.infrastructureAsCode.config;
        default: return '';
      }
    };

    const handleCopy = async () => {
      await navigator.clipboard.writeText(getRawContent(activeTab));
      setCopied(true);
      setTimeout(() => setCopied(false), 2000);
    };
    ```

    **Download button:**
    ```typescript
    const getFilename = (tab: string): string => {
      switch (tab) {
        case 'github': return 'github-actions.yml';
        case 'security': return 'security-scanning.yml';
        case 'infra': return `infrastructure.${pipelineConfig?.infrastructureAsCode.provider === 'terraform' ? 'tf' : 'yml'}`;
        default: return 'pipeline.yml';
      }
    };

    const handleDownload = () => {
      const content = getRawContent(activeTab);
      const url = URL.createObjectURL(new Blob([content], { type: 'text/plain' }));
      const a = document.createElement('a');
      a.href = url;
      a.download = getFilename(activeTab);
      a.click();
      URL.revokeObjectURL(url); // revoke immediately to prevent memory leak
    };
    ```

    **Buttons placement:** Top-right of the card, in the header row next to the tabs:
    ```tsx
    <div className="flex items-center gap-1">
      <Button size="sm" variant="ghost" onClick={handleCopy} className="text-slate-400 hover:text-slate-200">
        {copied ? <Check className="h-4 w-4 text-emerald-400" /> : <Copy className="h-4 w-4" />}
      </Button>
      <Button size="sm" variant="ghost" onClick={handleDownload} className="text-slate-400 hover:text-slate-200">
        <Download className="h-4 w-4" />
      </Button>
    </div>
    ```

    **Empty state:** Skeleton code block with gray placeholder lines — 6-8 lines of `<div className="h-3 bg-slate-700/50 rounded" />` with varying widths, matching existing skeleton pattern from Phase 1.

    **Graceful degradation:** If `analysisResult.pipeline` is null, show "Pipeline data unavailable — agent failed" with AlertTriangle.

    **Accept optional `compact` prop:** When `compact={true}` (used on overview page), reduce max-h to 200px and hide download button. When false/absent (used on pipeline page), full height.
