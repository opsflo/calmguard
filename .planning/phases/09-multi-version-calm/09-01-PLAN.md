## Goal

Build a pre-Zod normalization layer that detects CALM version from raw JSON, normalizes v1.0 documents to v1.1-compatible shape, preserves v1.2 extra fields, and carries the detected version through `ParseSuccess`. After this plan, `parseCalm()` accepts v1.0, v1.1, and v1.2 documents and returns a `version` field. No UI or store changes yet.

## Context

- **CONTEXT.md locked decisions:** Schema inference for version detection. Fill missing fields with defaults. Map old types (`apigateway` -> `service`, `microservice` -> `service`, `uses` -> `connects`). Lenient validation. Support v1.0, v1.1, v1.2.
- **Research:** Pre-Zod normalization pattern. Real v1.0 reference file analyzed: `calmSchemaVersion` field, `type` not `node-type`, `from`/`to`/`type` relationships, `name` as unique-id, `metadata.description`. v1.2 adds `adrs` (string array), `decorators`, `timelines`.
- **Current parser:** `parser.ts` calls `calmDocumentSchema.safeParse(json)` directly. `ParseSuccess` has `{ success: true, data: CalmDocument }`. `types.ts` has strict `nodeTypeSchema` enum and discriminated union for relationships.
- **Key pitfalls:** v1.0 uses `type` not `node-type`, `name` not `unique-id`, flat `{ from, to, type }` relationships. Zod strips unknown keys by default. Version detection order matters (v1.0 first, then v1.2, then default v1.1).

```typescript
export type { CalmVersion } from './normalizer';

// Extend calmDocumentSchema:
export const calmDocumentSchema = z.object({
  nodes: z.array(calmNodeSchema).min(1, 'Document must have at least one node'),
  relationships: z.array(calmRelationshipSchema),
  controls: z.record(z.string(), controlDefinitionSchema).optional(),
  flows: z.array(calmFlowSchema).optional(),
  // v1.2 additions — preserve without strict typing
  adrs: z.array(z.string()).optional(),
  decorators: z.unknown().optional(),
  timelines: z.unknown().optional(),
  // v1.0 legacy top-level fields — accept but don't require
  calmSchemaVersion: z.string().optional(),
  name: z.string().optional(),
});
```

**parser.ts changes:**
```typescript
import { detectCalmVersion, normalizeCalmDocument, type CalmVersion } from './normalizer';

export interface ParseSuccess {
  success: true;
  data: CalmDocument;
  version: CalmVersion; // NEW
}

export function parseCalm(json: unknown): ParseResult {
  const version = detectCalmVersion(json);
  const normalized = normalizeCalmDocument(json, version);
  const result = calmDocumentSchema.safeParse(normalized);
  if (result.success) {
    return { success: true, data: result.data, version };
  }
  return { success: false, error: formatZodError(result.error) };
}
```
