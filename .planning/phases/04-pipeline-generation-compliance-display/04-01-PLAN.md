  <action>
    Create `src/components/dashboard/compliance-card.tsx` as a `'use client'` component.

    **SVG Gauge (top section):**
    - Use a `<svg>` with viewBox centered on a circle. Radius = 56, circumference = 2 * Math.PI * 56 = ~351.86.
    - Background track: full circle with `stroke="currentColor"` in `text-slate-700`, `strokeWidth={8}`, `fill="none"`.
    - Foreground arc: same circle but with `strokeDasharray={CIRCUMFERENCE}` and `strokeDashoffset={CIRCUMFERENCE * (1 - displayScore / 100)}`. Apply `strokeLinecap="round"` for rounded ends. `transform="rotate(-90 64 64)"` to start from 12 o'clock position. Color via `getGaugeColor(displayScore)`:
      - score < 40 → `#ef4444` (red-500)
      - score < 70 → `#f59e0b` (amber-500)
      - score >= 70 → `#10b981` (emerald-500)
    - Add CSS `transition: stroke-dashoffset 0.1s linear` on the foreground arc so it smoothly follows the count-up.
    - Center text: `<text>` element showing `displayScore` in large bold font + "/ 100" subtitle below. Color matches gauge color.

    **Count-up animation hook (`useCountUp`):**
    - Accept `targetScore: number` and `duration: number = 1200`.
    - Return `displayScore: number` starting at 0.
    - Use `useEffect` + `requestAnimationFrame` loop:
      - Track `startTime` via `performance.now()`.
      - Each frame: `progress = Math.min((currentTime - startTime) / duration, 1)`.
      - Ease-out cubic: `eased = 1 - Math.pow(1 - progress, 3)`.
      - `setDisplayScore(Math.round(targetScore * eased))`.
      - Continue if `progress < 1`.
    - Cleanup: `cancelAnimationFrame(rafRef.current)` in useEffect return.
    - Only trigger when `targetScore` changes and is > 0.

    **Per-framework bars (below gauge):**
    - Read `analysisResult.risk.frameworkScores` from Zustand.
    - `frameworkScores` is an array of `{ framework: string, score: number, rating: string }`.
    - Render each as a row:
      - Label: framework name (left-aligned, `text-sm text-slate-300`)
      - Score: percentage (right-aligned, `text-sm text-slate-400`)
      - Progress bar: `div` with `bg-slate-700` track, inner `div` with colored fill and `transition-all duration-1000 ease-out` for the width.
      - Bar color: same logic as gauge color based on individual score.

    **Empty state (when no risk data):**
    - Show SVG gauge with dashed stroke (`strokeDasharray="4 4"`) in `text-slate-700`, no foreground arc.
    - Center text: "Run analysis to see score" in `text-slate-500`, small font.
    - Framework bars: skeleton-like gray bars with `bg-slate-700/50`.

    **Component structure:**
    ```
    <Card className="bg-slate-800 border-slate-700">
      <div className="p-6">
        <h3 className="text-sm font-medium text-slate-400 mb-4">Compliance Score</h3>
        {/* SVG Gauge */}
        <div className="flex justify-center mb-6">
          <svg width={128} height={128} viewBox="0 0 128 128">
            ...
          </svg>
        </div>
        {/* Overall rating text */}
        {riskData && (
          <p className="text-center text-sm text-slate-400 mb-6">
            Rating: <span className="font-medium text-slate-200">{riskData.overallRating}</span>
          </p>
        )}
        {/* Framework bars */}
        <div className="space-y-3">
          {frameworkBars}
        </div>
      </div>
    </Card>
    ```

    Read from Zustand store using individual selectors:
    ```typescript
    const riskData = useAnalysisStore((state) => state.analysisResult?.risk ?? null);
    const status = useAnalysisStore((state) => state.status);
    ```
