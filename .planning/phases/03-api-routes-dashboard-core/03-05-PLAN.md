    **src/components/graph/utils/calm-to-flow.ts** (NEW):
    Transform CALM data into React Flow nodes and edges.

    Function signature:
    ```typescript
    export function calmToFlow(
      analysisInput: AnalysisInput,
      architectureAnalysis: ArchitectureAnalysis | null,
      riskAssessment: RiskAssessment | null
    ): { nodes: Node[]; edges: Edge[] }
    ```

    Algorithm:
    1. **Create nodes from analysisInput.nodes:**
       - Map each CalmNode to a React Flow Node
       - `id`: node['unique-id']
       - `type`: Map CALM node-type to custom node type name (service, database, webclient, actor, system for system/ecosystem, default for others)
       - `data`: `{ label: node.name, description: node.description, complianceStatus: 'unknown', nodeType: node['node-type'] }`
       - `position`: `{ x: 0, y: 0 }` (dagre will set this)

    2. **Create edges from analysisInput.relationships:**
       - For `connects` relationships: Create edge from source.node to destination.node
       - `id`: relationship['unique-id']
       - `data`: `{ protocol: relationship.protocol || '' }`
       - `type`: 'protocol' (custom edge type)
       - For `interacts` relationships: Create edges from actor to each node
       - Skip `deployed-in` and `composed-of` (handled as trust boundaries)

    3. **Create trust boundary parent nodes** (if architectureAnalysis exists):
       - For each trustBoundary in architectureAnalysis.trustBoundaries:
         - Create a parent group node with type 'trustBoundary'
         - Set child nodes' `parentId` to this group node's id
         - Set child nodes' `extent` to 'parent'
       - IMPORTANT: Parent nodes MUST appear before children in the nodes array (per research pitfall #2)

    4. **Apply compliance coloring** (if riskAssessment exists):
       - For each node, check riskAssessment.nodeRiskMap for compliance status
       - Update node.data.complianceStatus accordingly
       - If no risk data: keep 'unknown'

    5. **Apply dagre layout** by calling `getLayoutedElements(nodes, edges, 'LR')`

    6. Return `{ nodes, edges }`

    Handle null architectureAnalysis gracefully — just skip trust boundaries and compliance coloring.
